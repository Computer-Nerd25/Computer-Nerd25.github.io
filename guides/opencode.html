OpenCode Custom Provider Setup Guide

Installation Steps

1. Download OpenCode

Install OpenCode CLI tool using the official method:

npm install -g opencode-ai

2. Navigate to .config/opencode Folder

Create and navigate to the configuration folder:

# Windows
mkdir %USERPROFILE%\\.config\\opencode
cd %USERPROFILE%\\.config\\opencode

# macOS/Linux
mkdir -p ~/.config/opencode
cd ~/.config/opencode

Enter the following command, select AQUA, and input your API Key:

Select AQUA provider and enter your API Key.
