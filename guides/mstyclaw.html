Msty Claw Agent Setup Guide

Installation Steps

1. Launch Msty Claw

Open the native Msty Claw Desktop application interface on your machine.

2. Open Providers in Settings

Click the Settings Cog (Gear Icon) in the lower-left tray and navigate into the Providers configuration view.

3. Add Custom OpenAI Compatible Provider

Locate the Custom OpenAI Compatible section, click Add / Enable, and name it Aqua Devs.

4. Set the Base URL

Change the Base URL parameter node precisely to our endpoint:

Base URL:  https://api.aquadevs.com/v1

5. Enter Your API Key

Provide your private token when prompted for the primary API Key string container box.

API Key sourced from your Aqua account dashboard

6. Add Your Model Slugs

Click Add Model within the block to input your preferred automation model slug targets (e.g. loading… or loading…) and hit save.

7. Start a Task or Playbook

Start a fresh Task or Playbook from the main dashboard workspace, select your Aqua model tier, point it at a scoped project directory folder, and deploy your background loop runner safely.

Complete

After configuration, you can start using AQUA provider models with Msty Claw.
