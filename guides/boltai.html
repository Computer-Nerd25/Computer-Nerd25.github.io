BoltAI Native Endpoint Configuration

Installation Steps

1. Open BoltAI Settings

Open BoltAI on your machine and press the standard macOS shortcut Cmd + , to trigger the main Settings sheet.

2. Add an OpenAI-Compatible Provider

Select the Models (or Providers) tab from the header menu, click the + (Add) icon, and select OpenAI-compatible Server from the dropdown selection list.

3. Set Friendly Name & Base URL

Change the service descriptor name to Aqua Devs and input the Base URL target precisely as:

Base URL:  https://api.aquadevs.com/v1
Friendly Name:  Aqua Devs

4. Enter Your API Key

Paste your private secret token from your account dashboard directly into the API Key text input box.

5. Register Model Slugs

Manually register your target model tier slugs (e.g. loading…, loading…, or loading…) along with your preferred context token caps.

6. Enable Streaming & Set Defaults

Ensure Streaming is toggled ON so your tokens render in real-time, click Save Changes, and set your new Aqua models as the default workspace assistants.

Complete

After configuration, you can start using AQUA provider models with BoltAI.
